<br>

<br>

## 설명

10초마다 CGV 용산아이파크몰 IMAX관의 예매 상황을 확인하여, 알림을 보내줍니다.

텔레그램에서 친구추가하여 예매 오픈시 알람을 받을 수 있습니다.

## 개발

-   Typescript
-   Node.js (Express.js)

## 권장 환경

-   Fedora 계열 Linux (권장 - Rocky Linux)
-   **.env**와 **src/data/chatId.json** 파일 설정 필수
